# Counter App – Step by Step

## Step 1: Create the Project Files

First, create three files:

* `index.html`
* `style.css`
* `script.js`

These files are used for the structure, design, and functionality of the app.

## Step 2: Build the HTML Structure

In the HTML file, create:

* A heading for the app title.
* A section to display the counter value.
* Three buttons:

  * **Increase (+)**
  * **Decrease (-)**
  * **Reset**

This is the basic layout of the application.

## Step 3: Style the App with CSS

Use CSS to:

* Center all the content on the page.
* Add colors, spacing, and fonts.
* Style the buttons.
* Make the app look clean and user-friendly.

## Step 4: Create the Counter Variable

In JavaScript, create a variable to store the current counter value. Initially, set it to **0**.

## Step 5: Update the Counter Display

Create a function that updates the number shown on the screen whenever the counter value changes.

## Step 6: Increase the Counter

Create a function that increases the counter value by **1** and then updates the display.

## Step 7: Decrease the Counter

Create a function that decreases the counter value by **1** and then updates the display.

## Step 8: Reset the Counter

Create another function that sets the counter value back to **0** and updates the display.

## Step 9: Connect the Buttons

Link each button to its corresponding JavaScript function:

* **Increase** button → Increase the counter.
* **Decrease** button → Decrease the counter.
* **Reset** button → Reset the counter.

## Step 10: Test the App

Open `index.html` in your browser and test all three buttons to make sure the counter works correctly.

## Conclusion

This project helps beginners understand how **HTML** creates the structure, **CSS** improves the design, and **JavaScript** adds functionality by updating the counter based on user interactions.
